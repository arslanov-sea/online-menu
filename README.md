# Онлайн-меню

Веб-приложение для отображения меню заведения общепита. Написано на Django (Python).

## Что делает

На главной странице выводится список блюд, сгруппированных по категориям (например: пицца, напитки и т.д.). Каждое блюдо имеет название, описание, цену и фотографию. Категории выводятся в порядке приоритетности — это можно настроить через административную панель.

## Из чего состоит

- Модель `Type_d` — тип (категория) блюда. У каждого типа есть название и числовой приоритет, который определяет порядок отображения.
- Модель `Dish` — блюдо. Привязано к категории. Содержит название, описание, основную цену, дополнительную цену (необязательно) и изображение.
- Страница меню отображает блюда, сгруппированные по категориям.
- Административная панель Django позволяет добавлять, удалять и редактировать блюда и категории.

## Стек

- Python 3
- Django
- SQLite
- HTML, CSS, JavaScript

## Запуск

1. Установить зависимости:
   ```
   pip install django pillow
   ```

2. Применить миграции:
   ```
   python manage.py migrate
   ```

3. Создать суперпользователя для доступа к админке:
   ```
   python manage.py createsuperuser
   ```

4. Запустить сервер:
   ```
   python manage.py runserver
   ```

5. Открыть в браузере `http://127.0.0.1:8000/`

Административная панель доступна по адресу `http://127.0.0.1:8000/admin/`.
